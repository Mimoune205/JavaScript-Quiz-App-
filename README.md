# JavaScript Exercises & Quiz App

## 📖 Overview

This project contains three simple JavaScript exercises accessible through buttons in a web page:

* **Exercice 01**: Performs basic arithmetic operations (sum, difference, product, quotient)
* **Exercice 02**: A number guessing game where the user tries to find a random number
* **Exercice 03 (Quiz)**: A short quiz about world capitals

## 🚀 Features

* Interactive buttons
* User input via `prompt()`
* Feedback via `alert()`
* Simple and beginner-friendly code structure

## 🛠️ Technologies Used

* HTML5
* JavaScript (Vanilla)

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. Open `index.html` in your browser

## 📚 Learning Goals

* Understand JavaScript functions
* Practice loops and conditionals
* Work with user input and output

## ⚠️ Notes

This project uses `prompt()` and `alert()` for simplicity. For a better user experience, these could be replaced with custom UI elements in the future.
